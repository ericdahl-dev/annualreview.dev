Privacy

AnnualReview.dev is designed to work from the evidence you provide. This project links annual review output back to GitHub evidence and is intended to minimize unnecessary data retention.

What the app uses

  • GitHub data that you fetch through the app or paste into it.
  • Optional goals or review context that you add yourself.
  • Basic product analytics when enabled by the deployment.

What the app does not claim

  • It does not claim ownership of your code or pull requests.
  • It does not invent performance outcomes on your behalf.
  • It does not require an account just to view the marketing site.

If you run the project yourself, your deployment choices determine how evidence is stored, logged, or retained.
